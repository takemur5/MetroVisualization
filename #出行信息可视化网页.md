# 地铁站点圆圈功能说明

## 1. 外圆（站点总览）
- **绘制逻辑**：
  - 根据站点作为起点和终点的次数，决定外圆的颜色（渐变填充）和描边颜色。
  - 外圆半径与总使用次数成正比，最小半径为12px。
  - 外圆带有阴影效果（drop-shadow）。
- **交互功能**：
  - 鼠标悬停时：
    - 显示该站点所有乘客的出行信息（包括姓名、角色、路线）。
    - 外圆高亮（半径增大、填充色变为纯色）。
  - 鼠标移出时：
    - 恢复原始样式，隐藏信息弹窗。
  - 鼠标点击时：
    - 弹窗展示该站点所有乘客的详细信息（姓名、角色、出生年份、职业、出行类型、完整路线）。

## 2. 内圆（单个乘客出行）
- **绘制逻辑**：
  - 仅当该站点被多于1位乘客使用时绘制。
  - 最多显示5个内圆，半径依次递增（如4、10、16、22、28）。
  - 内圆样式与外圆一致（渐变填充、无描边、带阴影）。
- **交互功能**：
  - 鼠标悬停时：
    - 显示对应的圆的出行信息
    - 对应的圆高亮（可设置描边高亮）。
  - 鼠标移出时：
    - 恢复原始样式，隐藏信息弹窗。
  - 鼠标点击时：
    - 可扩展为显示该乘客的详细出行信息。

## 3. 信息弹窗（infoPopup）
- **显示内容**：
  - 站点名、乘客姓名、年龄（2025-出生年份）、是否出生于深圳、抵深时间、身份（职业）、出行类型、完整路线等。
- **位置与样式**：
  - 弹窗跟随鼠标位置，带透明背景和圆角。

## 4. 代码规范
- 代码结构清晰，便于扩展和维护。

## 5. 其它主要功能说明
### 5.1 地铁线路绘制
- 加载 shenzhen_lines.geojson ，用 D3 在 SVG 层绘制所有地铁线路，统一为灰色线条，带透明度。
- 鼠标悬停在线路上时，弹出提示框显示线路名称及起止站点。
### 5.2 地铁站点数据处理
- 加载 shenzhen_stations.geojson ，将站点名称与地理坐标建立映射，便于可视化和交互。
### 5.3 乘客出行数据处理
- 加载 interview.csv ，解析每位乘客的出行路径，统计每个站点作为起点和终点的次数。
- 记录每位乘客的基本信息（出生年份、是否深户、来深年份、职业等），并与其出行路径关联。
### 5.4 出行路径可视化
- 根据乘客出行数据，绘制每一条出行路径为带有渐变色的曲线路径（绿色为起点，红色为终点）。
- 鼠标悬停在路径上时，显示该乘客的出行起止站点信息。
### 5.5 渐变与高亮效果
- 为每条路径动态生成线性渐变色，提升可视化辨识度。
- 起点和终点圆圈分别使用径向渐变（绿色/红色）高亮。
### 5.6 地图交互与自适应
- 基于 Leaflet 地图，支持缩放、拖动等交互。
- 地图移动或缩放时，自动更新所有 SVG 元素（圆圈、路径、渐变等）的位置，确保与底图对齐。
### 5.7 信息弹窗与提示
- 鼠标悬停在站点圆圈、路径、地铁线路等元素上时，均会弹出信息提示框，显示对应的详细信息。
- 信息弹窗始终显示在最上层，避免被其他元素遮挡。
### 5.8 模式切换
- 支持工作日与节假日两种模式切换，分别加载和展示不同的数据行，动态刷新所有可视化内容。
